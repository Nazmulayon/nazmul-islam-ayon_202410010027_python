# Week 5 Documentation

## 1. Problem Statement
Build a Python program to calculate a customer's cafe bill automatically.

## 2. Inputs
- Customer name
- Coffee quantity
- Tea quantity
- Sandwich quantity

## 3. Outputs
- Customer receipt
- Total amount (RM)

## 4. Process Flow
1. Get customer name.
2. Get item quantities.
3. Calculate total bill.
4. Print receipt.

## 5. Constraints
- Quantity must be a whole number.
- Quantity cannot be negative.

## 6. Problem Decomposition
- Input customer details.
- Calculate total.
- Display receipt.
